# 🌍 TripToGo - Travel Website

## 📖 Overview

TripToGo is a responsive travel website built using **HTML5** and **CSS3**. The website showcases travel destinations, tour packages, honeymoon packages, and premium group tours with a modern glassmorphism hero section and attractive card-based layout.

---

## ✨ Features

* 🏠 Modern Navigation Bar
* 🌄 Full-screen Hero Section
* 🔍 Destination Search Section
* 🎒 Tour Package Categories
* 💍 Wedding & Honeymoon Packages
* 🌏 International Tour Packages
* ⭐ Premium Group Tours
* 🖼️ Beautiful Image Gallery
* 📱 Responsive Design
* 🌐 Social Media Footer

---

## 🛠️ Built With

* HTML5
* CSS3
* Font Awesome Icons

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.

No additional installation is required.
